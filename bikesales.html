Bike Sales

This is the analysis of a bike sales dataset using Tableau to design and create a dashboard for insight visualisation.

Main objectives

The idea behind this project is to curate the raw data from an excel file, organize it and display relevant metrics in a visual dashboard, allowing for better understanding of some Key Performance Indicators, and hopefully allowing the user to derive sales insights from the - now comprehensive - data!

Top 10 Products

This is one of the components of the main dashboard and it showcases the top 10 products by order quantity.

Profit By Subcategory

This is a component created to show the profit by product subcategory, using a stacked chart separating male and female clients.

Profit by Country

This is an example what is possible with data visualization. This map represents the profits of bike sales by country using a bubble to depict the amount of sales.

Main Dashboard

This is the dashboard that aggregates all the other components (and some more that were not mentioned) and brings the data to life, allowing for information to be viewed for multiple angles for insight generation. This is where data turns into insights, turns into ideas.
